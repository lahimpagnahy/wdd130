Overview

Purpose

The purpose of this rafting website is to provide an online platform for potential customers to learn about the rafting services offered, book trips, and get information about safety, equipment, and the unique rafting experiences available. The website aims to increase the company's visibility, attract more customers, and provide a convenient way for users to engage with the business and make reservations.

Audience

The target audience for the rafting website includes adventure enthusiasts, families looking for outdoor activities, and corporate groups seeking team-building exercises.

Style Guide

Color Palette

Our rafting website uses a vibrant and nature-inspired color palette to evoke the spirit of adventure and the great outdoors. The primary colors include Deep Blue (#003366) for trust and stability, Vibrant Green (#009933) for nature and energy, and Earthy Brown (#996633) for reliability. Accent colors include Bright Orange (#FF6600) for calls to action and Sky Blue (#66CCFF) for a fresh, open fee

Typography

Headings Example Font

We use "Montserrat" for headings to create a bold and adventurous look.

Normal Text/Paragraph Example Font

For body text, we use "Roboto" for a clean and readable experience. Example text - Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus lacinia odio vitae vestibulum vestibulum.

Colored Callout Example Font

Callouts and special text use the same "Roboto" font but in a highlighted color for emphasis. Example text - Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus lacinia odio vitae vestibulum vestibulum.
